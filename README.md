# Over Heating Detection and Monitoring System

## Overview
This project is a PCB-based Over Heating Detection and Monitoring System developed to monitor temperature levels and detect overheating conditions in electronic systems.

The system continuously reads temperature data from sensors and provides alerts when the temperature exceeds a predefined threshold.

## Features
- Real-time temperature monitoring
- Overheating detection
- Alert/notification mechanism
- Custom PCB design
- Arduino-based implementation
- Compact and efficient system

## Technologies Used
- Arduino
- Embedded C/C++
- PCB Design
- Temperature Sensors

## Files Included
- Arduino source code
- PCB board design (.brd)
- Schematic files (.sch)

## Applications
- Industrial safety systems
- Electronic device monitoring
- Automation projects
- Temperature protection systems
